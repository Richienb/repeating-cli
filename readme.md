# repeating-cli [![Build Status](https://travis-ci.org/sindresorhus/repeating-cli.svg?branch=master)](https://travis-ci.org/sindresorhus/repeating-cli)

> Repeat a string


## Install

```
$ npm install --global repeating-cli
```


## Usage

```
$ repeating --help

  Usage
    $ repeating <string> <count>

  Example
    $ repeating 'unicorn ' 2
    unicorn unicorn 
```
